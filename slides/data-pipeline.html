2
The gender lookup table
speaker_genders.csv (new)
Built a verified gender for every unique speaker using a layered approach — most reliable sources first.
1 Church Historian's Press — official list of women speakers + web confirmations 66 women
2 Web-verified overrides — names the library got wrong (Marion D. Hanks = male, etc.) 7 fixed
3 gender_guesser library — Python library that classifies first names 527 classified
4 Calling-based fallback — for names the library doesn't recognize 153 remaining
